# 📝 Next Word Prediction using LSTM

This project is a Streamlit web app that predicts the next word in a sentence using an LSTM model.

## 🚀 Features
- Predict next word from input text
- Built with LSTM (Deep Learning)
- Simple and interactive UI using Streamlit

## 📁 Files
- `app.py` → Streamlit app
- `lstm_model.h5` → trained model
- `tokenizer.pkl` → tokenizer
- `max_sequence_len.pkl` → sequence length

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
